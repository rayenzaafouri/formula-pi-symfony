⚠️ Note on environment variables (Windows systems)

After completing the installation of XAMPP, add the path of your PHP interpreter to the system environment variables:

  • Go to the XAMPP directory and copy the location of your PHP interpreter: C:\xampp\php
  • Navigate to Control Panel → System → Advanced system settings → Environment Variables.
  • In System Variables, select the Path variable, then click Edit and paste the PHP interpreter path: C:\xampp\php
  • To test it in the Command Prompt (CMD), type php -v and ensure you have the correct PHP version.
